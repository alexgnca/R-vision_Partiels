1. Les Fondamentaux de la Stratégie
Définitions Clés[cite: 36, 37]
  • Origine : Du grec stratos (armée) et argos (conduire)[cite: 36].
  • Stratégie (Chandler) : Détermination des buts à long terme et allocation des ressources pour les atteindre[cite: 37].
  • Marketing (Kotler) : Satisfaire le client de façon rentable et durable (via les 4P)[cite: 37].
Méthodes d'Amélioration[cite: 35, 36]
  • PDCA (Roue de Deming) :
    - Plan (Planifier)
    - Do (Développer/Mettre en œuvre)
    - Check (Contrôler les écarts)
    - Act (Agir/Ajuster)[cite: 36].
L'Environnement VUCA : Pour décider dans un monde complexe[cite: 36].
→ Volatility (Fréquences des changements)
→ Uncertainty (Difficultés de prévision long terme)
→ Complexity (Quantité de paramètres)
→ Ambiguity (La solution miracle n'existe pas)[cite: 36].

2. L'Analyse Stratégique (Matrices & Modèles)
Du SWOT au TOWS[cite: 36, 39]
  • SWOT : Bilan interne (Forces/Faiblesses) et externe (Opportunités/Menaces)[cite: 37].
  • Le TOWS croise ces dimensions pour créer 4 stratégies :
    - SO (Maxi-Maxi) : Stratégie d'attaque. Utiliser les forces pour saisir les opportunités[cite: 36, 39].
    - WO (Mini-Maxi) : Corriger les faiblesses grâce aux opportunités[cite: 36, 39].
    - ST (Maxi-Mini) : Stratégie défensive. Utiliser les forces pour minimiser les menaces[cite: 36, 39].
    - WT (Mini-Mini) : Stratégie de survie/retrait[cite: 36, 39].
La Matrice BCG[cite: 37]

Croise la Croissance du marché et la Part de marché relative.

  • Star (Vedette) : Forte croissance, forte PDM (Besoin de cash)[cite: 37].
  • Vache à lait : Faible croissance, forte PDM (Génère le cash)[cite: 37].
  • Dilemme : Forte croissance, faible PDM[cite: 37].
  • Poids mort : Faible croissance, faible PDM[cite: 37].
Stratégies de Porter[cite: 36, 39]
  • Domination par les coûts : Cibler tout le marché avec les coûts les plus faibles (effet d'expérience)[cite: 36, 39].
  • Différenciation : Cibler tout le marché avec un caractère unique perçu[cite: 36, 39].
  • Concentration (Focalisation/Niche) : Cibler un segment restreint[cite: 36, 39].
Matrice d'Ansoff[cite: 37, 39]
  • Pénétration : Produit Existant / Marché Existant[cite: 37, 39].
  • Extension de marché : Produit Existant / Marché Nouveau[cite: 37, 39].
  • Développement produit : Produit Nouveau / Marché Existant[cite: 37, 39].
  • Diversification : Produit Nouveau / Marché Nouveau[cite: 37, 39].

3. Le Marketing Stratégique
La Méthode SCP (Segmenter - Cibler - Positionner)[cite: 36, 39]
  • Segmenter : Découper le marché en groupes homogènes (constat du marché)[cite: 36, 39].
  • Cibler : Décision stratégique du ou des segments visés[cite: 36, 39].
    - Indifférencié (même produit pour tous).
    - Différencié (offre spécifique par segment).
    - Concentré (un seul segment).
    - One-to-one (offre sur-mesure)[cite: 36, 39].
  • Positionner : Place unique dans l'esprit du consommateur (Sortir de l'anonymat, ex: le poisson rouge coloré)[cite: 36, 39].
Stratégie Océan Bleu (CERE) : Créer - Élever - Réduire - Éliminer. Permet d'améliorer sa proposition de valeur pour rendre la concurrence obsolète[cite: 39].

4. Action Commerciale, Négociation et PNL
Le Comportement d'Achat[cite: 36, 37]
  • Les 5 étapes de décision : 1. Reconnaissance du besoin → 2. Recherche d'info → 3. Évaluation → 4. Achat → 5. Évaluation post-achat[cite: 36, 37].
  • Le SONCASER (Motivations) : Sécurité, Orgueil, Nouveauté, Confort, Argent, Sympathie + Environnement, Région[cite: 36].
Méthodes de Vente[cite: 39]
  • Méthode ADAC : Accueil, Découverte, Argumentation, Conclusion[cite: 39].
  • Le Sous-Marin : Rapport de confiance → Contrat préalable → Douleur → Budget → Décision → Conclusion → Après-vente[cite: 39].
Fidélisation & Churn[cite: 39]
  • Loi de Pareto : 20% des clients font 80% des ventes[cite: 39].
  • Churn : Attrition (perte de clients)[cite: 39].
  • Stratégies : Ventes incitatives (Up-sell vers gamme supérieure), Ventes croisées (Cross-sell)[cite: 39].
Communication & PNL[cite: 39]
  • "On ne peut pas ne pas communiquer." 60% du message est Non-Verbal (posture, voix)[cite: 39].
  • L'Index de Référence (PNL) :
    1. Processus Interne (Penser)
    2. État Interne (Sentir : états de ressource/positifs ou états limitants)
    3. Comportement Externe (Agir)[cite: 39].
⚠️
Pièges classiques QCM
Attention aux confusions dans les Matrices !
  • BCG : Ne confonds pas "Poids Mort" (faible croissance/faible PDM) avec "Dilemme" (Forte croissance/Faible PDM)[cite: 37].
  • TOWS vs SWOT : Le SWOT fait le bilan, le TOWS croise les données pour créer les stratégies (SO, WO, ST, WT)[cite: 36, 39].
  • SONCASER : Les deux nouvelles lettres (E et R) signifient Environnement et Région[cite: 36].
  • PDCA : Le "D" veut dire Do (Développer/Mettre en œuvre), pas Decide ni Discover[cite: 36].
